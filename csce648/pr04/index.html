1 Theory

Definition 1.1. For space $E$(total space), $B$(base), $F$(fiber) and surjection \begin{equation}\pi: E\rightarrow B,\end{equation} if for $\forall x\in E$, there exists an open neighborhood U of $\pi(x)$ such that $\pi^{-1}$(U) and U$\times$F are homeomorphic, then (E,B,F,$\pi$) is called a fiber bundle. And E has the form of product space B$\times$F.

Definition 1.2.{$U_i$} is an atlas of an $n$-dimensional manifold $U$. For each chart $U_i$, define \begin{equation}\phi_i: U_i\rightarrow\mathbb{R}^n\end{equation}

If $U_i\cap U_j\neq \emptyset$, define \begin{equation}\varphi_{ij}=\phi_j\circ\phi_i^{-1}\end{equation}

And if $\varphi_{ij}$ has the form of \begin{equation}\varphi_{ij}=(\varphi_{ij}^1(x),\varphi_{ij}^2(x, y))\end{equation} \begin{equation}\varphi_{ij}^1: \mathbb{R}^{n-p}\rightarrow\mathbb{R}^{n-p}\end{equation} \begin{equation}\varphi_{ij}^2: \mathbb{R}^{n}\rightarrow\mathbb{R}^{p}\end{equation} then for any stripe $s_i:x=c_i$ in $U_i$ there must be a stripe in $U_j$ $s_j:x=c_j$ such that $s_i$ and $s_j$ are the same curve in $U_i\cap U_j$ and can be connected. The maximal connection is a leaf.

This structure is called a $p$-dimensional foliation $F$ of an $n$-dimensional manifold $U$.

Property 1.1. All the fiber bundles compose a group $G_F=(\{F_i\},\circ)$.

If we regard the fiber bundle(1-manifold) as a parameterization on a surface(2-manifold), we can prove it easily.

For example, we use the parameterization on a torus. \begin{equation}\{u,v\}\rightarrow\{(R+r \cos v)\cos u, (R+r \cos v)\sin u, r \sin v\}\end{equation}

Then we can get the trivial fibers in Fig. 2.

\begin{equation}\{u,v\}=\{t,const\}\end{equation}

To generalize the creating method of fiber bundle, it's necessary to define the transformation functions.

Definition 1.3. A transformation function $F:[0,1]^2\rightarrow[0,1]^2$ for fiber bundle must be:

  • $F$ is $C^1([0,1]^2)$
  • $F$ is bijection
  • $F(u,0)=F(u,1)$, $F(0,v)=F(1,v)$
    $F_u(u,0)=F_u(u,1)$, $F_u(0,v)=F_u(1,v)$
    $F_v(u,0)=F_v(u,1)$, $F_v(0,v)=F_v(1,v)$

An example is $F(u,v)=\{u,v+\frac{u}{2}\}$. The $\{\cdot\}$ inside means the fractional part. Generally, transformation functions always use $\{\cdot\}$ to make sure that $F$ have the properties listed above. In the following parts, we will omit it.

With transformation functions, we can easily get any fiber in any fiber bundle: \begin{equation}F:\{u,v\}=\{t,const\}\rightarrow\{u',v'\}=F(u,v)\rightarrow\{(R+r \cos v')\cos u', (R+r \cos v')\sin u', r \sin v'\}\end{equation}

Property 1.2. For $G_F$, we have following properties:

  • For $\forall F_i$ and $F_j$, $F_i\circ F_j$ is defined as \begin{equation}F_j\circ F_i:\{u',v'\}=F_j(F_i(u,v))\rightarrow\{(R+r \cos v')\cos u', (R+r \cos v')\sin u', r \sin v'\}\end{equation}
  • Identity element is \begin{equation}I:I(u,v)=\{u,v\}\end{equation}
  • $\forall F_i$, \begin{equation}F_i^{-1}:\{u',v'\}=F_i^{-1}(u,v)\rightarrow\{(R+r \cos v')\cos u', (R+r \cos v')\sin u', r \sin v'\}\end{equation}
